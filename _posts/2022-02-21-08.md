
</pre>

```python
df.iloc[4,2] # 5번 학생의 키 데이터 [4번째 위치의 2번째[키] 데이터]
```

</pre>

```python
df.iloc[[0,1],2] # 0,1번째 위치의 학생의 2번째[키] 데이터
```

2번    184
Name: 키, dtype: int64
</pre>

```python
df.iloc[[0,1], [3,4]] # 0,1번째 위치 학생의 3,4번째 데이터 [국어, 영어]
```
